# 파일정보

파일의 목록 또는 정보를 보고 싶다면 ls 명령어를  이용한다.

~~~
# ls -l sample.txt   // 아래의 정보를 보여준다. 
-rwxr--r-- 1 root root 0 7월 15 16:11 sample.txt
~~~

## 파일 유형

파일 유형은 아래의 표처럼 총 5가지이다.

<table>
    <tr>
    	<th>d</th>
        <td>디렉터리</td>
    </tr>
    <tr>
    	<th>-</th>
        <td>파일</td>
    </tr>
    <tr>
    	<th>b</th>
        <td>블록 디바이스</td>
    </tr>
    <tr>
    	<th>c</th>
        <td>문자 디바이스</td>
    </tr>

    	<th>l</th>

    </tr>
</table>
